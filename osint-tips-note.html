OSINT 小技巧笔记

发表于 2022-04-04 更新于 2023-11-21 本文字数： 843 阅读时长 ≈ 1 分钟
有关 OSINT 一些信息搜集技巧或者关键词的简要笔记
  • 搜索某国的外交记录，可以谷歌使用关键字 list of international trips
  • 搜索衣服的时候，可以只截取人物的上半身或者下半身，或者一些特征点
  • 图片搜索引擎中，公认最好的是 www.yandex.ru 和 www.google.com
  • 使用 yandex 可以搜索到更多关于欧洲和俄罗斯的信息；使用 google 可以搜索到更多关于美国和西方国家的信息；使用 yahoo 可以搜索到关于亚洲小国的更多信息。
  • 跟代码相关的点可以优先使用 github、gitee 和 coding 等代码托管平台
  • 跟社交相关的点可以优先使用 google、twitter 和 facebook
  • 想要查找被删除的互联网网页，或者查找被删除的社交网页，可以查找网页快照或者使用互联网档案：https://archive.org/
  • https://bpa.st/，一个将 python 代码转换为短 URL 的网站
  • https://www.duplichecker.com/reverse-image-search.php 一个反向图像搜索器，结合多个图片搜索引擎
  • https://what3words.com/ 使用三个词表示一个地点
  • https://www.shodan.io/ 一个可用于搜索互联网的搜索引擎，它可以搜索所有连接到 Internet 上的东西，包括红绿灯和粒子回旋加速器。
  • SNMP 协议的端口通常为 161 或者 162
  • Very-small-aperture terminal 简称 VSAT
  • linkedin 领英，可以搜索求职者的一些信息，包括教育经历和求职经历
  • 坐标反查网站：https://api.map.baidu.com/lbsapi/getpoint/index.html
  • 坐标转换网站：https://www.17ditu.com/tool/convert/
  • 免费恶意软件分析服务：https://www.virustotal.com/gui/home/upload，会显示软件运行过程https://www.joesandbox.com/#windows
