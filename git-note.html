使用SSH方式远程连接Github

打开Git Bash(使用命令行也可以)

先在本地生成对应邮箱的SSH RSA公钥密钥

使用默认生成路径，默认空key

前往 ~/.ssh 目录下寻找密钥（其中默认id_rsa是密钥信息，id_rsa.pub为公钥信息）

在 Github 中新建 SSH keys (Github - Settings - SSH and GPG keys - New SSH key)

Title 任意，Key 填入公钥信息 (id_rsa.pub 文件内容)
