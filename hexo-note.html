hexo 配置详解

hexo 博文头 - Front-matter

参数 意义 默认值
layout 排版 config.default_layout
title 标题 文件名
date 发布日期 文件创建日期
updated 更新日期 文件更新日期
comments 简介 true
tags 标签
categories 分类
permalink 永久链接的相对路径 (如/index.html) null
excerpt 纯文本摘录 (需要使用插件Post-Excerpt)
disableNunjucks 是否禁用 Nunjucks 的宏 ({{ }} / {% %}) 和 tag plugins 插件 false
lang 指定语言 _config.yml
